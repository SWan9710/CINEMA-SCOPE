#### 목표

1. 사용자에게  인기순, 신작순, 장르별 인기순 원하는 영화를 추천.

2. Tinder를 활용하여 랜덤으로 원하는 영화를 swipe를 통해 선택 할 수 있게 함.

3. 검색을 통해 원하는 영화에 대한 정보를 제공
